# CrochetCodex

A standalone crochet chart editor built as a browser-based HTML app with configuration files under `config/`.

[钩针图案绘制器](https://ricewithoutice.github.io/CrochetCodex/index.html)

## Project structure

- `index.html` contains the page markup and static asset references.
- `styles/app.css` contains the application styling.
- `scripts/app.js` contains the editor interaction and rendering logic.
- `config/i18n.js` contains language and label configuration.
